== Description ==
Latest Posts MultiSite Widget is a wordpress plugin that enables admin to add widgets which display latest posts content from other sites in a wordpress network. The widget supports custom post types, post images, excerpts and also custom fields data. Admin can use default template or make their custom template. 


== Installation ==

* Download "latest-posts-multisite.zip" to your computer.
* Login as administrator
* Click on Plugins --> Add New
* On "Install Plugins" page, clik on "Upload" 
* Browse and select "latest-posts-multisite.zip" and click on "Install Now"
* You can activate it now
* Finally, add the widget to your sidebar through the 'Appearance > Widgets' menu
* Go to your widget and adjust the settings
* Use template tags to make your custom tempplate : {title}, {link}, {excerpt}, {image}, .. and any custom field data %%custom_field_key%%


== Frequently Asked Questions ==

= How to add custom field data to template =

Just use custom field key and "%%" symbol, ex: %%custom_field_key%%
